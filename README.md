# SFSystem0.2

Internal frozen seafood inventory and sales management web app.

## Run

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the server:
   ```bash
   npm start
   ```
3. Open `http://localhost:3000`.

## Accounts

- Owner: owner@sfseafood.local / Owner123!
- Operator: operator@sfseafood.local / Operator123!

## Features

- Secure login with owner/operator roles
- Dashboard with inventory and sales summaries
- Product management and stock details
- Customer management
- Sales transaction entry and listing
- Stock-in and stock-out recording
- Stock movement history and filters
- Owner reports for daily, weekly, and monthly sales
